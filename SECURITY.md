# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 2026.x | ✅ Yes |
| < 2026 | ❌ No |

## Reporting a Vulnerability

Security is paramount in the MiniLam ecosystem. If you discover a vulnerability, please report it via the following steps:
1. **Email**: Send a detailed report to `ibnaratul.2025@gmail.com` (Project lead).
2. **Detail**: Include environment details, steps to reproduce, and potential impact.
3. **Reward**: We value transparency and will acknowledge all valid security contributions.

Maintain the veil. Stay secure.
